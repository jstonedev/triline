# Triline

## How to use

### 1. Clone repository

```sh
git clone https://github.com/jstonedev/triline
```

### 2. Run the app

Inside the root directory `triline`:

```sh
yarn install
yarn start
```

You can now open your browser and use the app on [http://localhost:3000](http://localhost:3000).
